# 2048-made-with-pygame
就是一個普通的2048

## What's new
1. 計算移動次數，在死亡時顯示。
2. 增加返回上一步功能。
3. 調整ui、調整動畫顯示。
4. 修改配色。
5. 拆分檔案手機版(2048.py)、電腦版(2048-pc.py)。
6. 調整螢幕顯示方法，就算螢幕放大也不會卡，但解析度會偏低。

* 手機版會盡可能符合螢幕大小，有虛擬方向鍵。
* 電腦版可以選擇尺寸(360, 480, 560, 720, 1080, 1440, 2160)。
* ~~無法保證手機上運行電腦版或電腦運行手機版會發生什麼。~~
* 手機執行會有導覽列或相機開孔的影響導致遊戲無法正常全螢幕顯示，記得程式碼233行NAV_BAR自己調整到不會出現黑邊，嘻嘻。
